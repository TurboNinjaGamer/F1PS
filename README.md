# F1PS (Formula Per Second)

Full-stack web aplikacija za prikaz real-time i istorijskih Formula 1 podataka korišćenjem OpenF1 API-ja.

## Tech stack
- React (client)
- Node.js + Express (server)
- MySQL (XAMPP) – korisnici i podešavanja profila (uskoro)

## Run locally
### Server
cd server
npm install
npm run dev

### Client
cd client
npm install
npm start
